# 팀프로젝트 CRUD


## 개요
JSP, Oracle을 이용하여 기본 CRUD 홈페이지 구현
## 개발 환경
java 1.8 (jdk 1.8) <br>
Oracle <br>
apache-tomcat-8.5.78 <br>
## DB 구조/구성
1. 회원 관리 <br>
- 신규 회원 가입 <br>
- 로그인 / 로그아웃 <br>
2. 게시판 관리 (QnA, Review)
- 글 작성 / 조회 <br>
- 글 수정 / 삭제 <br>
